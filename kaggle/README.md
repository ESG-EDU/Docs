## (1) Kaggle의 기본 개념

> Kaggle은 한마디로 `데이터 과학의 GitHub` 라고 할 수 있습니다.
데이터 과학 프로젝트를 올리고, 다른 사람들과 공유하고, 서로 경쟁하거나 협력하는 곳입니다.

### 주요 기능 요약

| `구분` | `설명` |
| :---: | :---: |
| `Competitions` | 기업이나 기관이 제공한 데이터를 이용해 주어진 문제를 해결하는 **경진대회** |
| `Datasets` | 전 세계 사용자들이 올린 **공개 데이터셋** 저장소 |
| `Notebooks` | Python / R 코드를 작성하고 실행할 수 있는 **클라우드 기반 Jupyter Notebook 환경** |
| `Code` | Notebook 기반으로 다른 사람의 코드를 **복사(Fork)** 해서 수정, 실험 가능 |
| `Community` | 질문/토론, 튜토리얼, 블로그 등을 공유하는 커뮤니티 |
| `Courses` | 머신러닝, 데이터 분석 기초를 배울 수 있는 **무료 교육 과정** |

## (2) Kaggle Competitions (경진대회)

> Kaggle의 핵심이자 가장 인기 있는 기능입니다.

- 기업이나 기관(예: Google, NASA, Facebook, WHO 등)이 문제를 내고 `데이터셋을 제공` 합니다.
- 참가자들은 모델을 개발하여 `예측 결과를 제출(submit)` 합니다.
- Kaggle은 제출된 결과를 `리더보드(Leaderboard)` 로 순위를 매깁니다.
- 상금이 걸린 대회도 많습니다 (수천~수십만 달러 규모).

## (3) Datasets (데이터셋)

- 전 세계 사용자들이 **CSV, JSON, 이미지 등 형태로 업로드**한 데이터셋을 공유합니다.
- `100,000개 이상의 데이터셋`이 존재하며 대부분 `무료로 다운로드 가능` 합니다.
- 예:
  * Titanic (생존자 예측)
  * Netflix 영화 평점 데이터
  * World Happiness Report

> 데이터셋은 개인 프로젝트나 연구용으로 매우 유용합니다.

## (4) Notebooks (코드 환경)

> Kaggle은 **Jupyter Notebook**을 클라우드에서 바로 실행할 수 있게 제공합니다.

- Python, R 지원
- GPU / TPU 무료 제공 (제한된 시간 내)
- TensorFlow, PyTorch, scikit-learn 등 주요 ML 라이브러리 기본 탑재
- 로컬 환경 설정 없이 바로 실행 가능

> “Titanic 데이터셋 분석” 노트북을 열고 `코드 실행 → 시각화 → 제출`까지 모두 브라우저에서 가능합니다.

## (5) Courses (무료 교육 과정)

> Kaggle의 `Learn` 메뉴에서는 **데이터 과학 입문자를 위한 짧고 실용적인 무료 강의**를 제공합니다.

| `코스명` | `내용` |
| :---: | :---: |
| `Python` | 파이썬 기본 문법 및 데이터 처리 |
| `Pandas` | 데이터 프레임 처리 |
| `Machine Learning` | 기본적인 ML 개념 |
| `Deep Learning` | 신경망의 기초 |
| `SQL` | 데이터베이스 질의 |
| `Data Visualization` | 데이터 시각화 기초 |

> 각 코스는 `이론 + 실습 + 퀴즈` 형태로 구성되어 있으며, 매우 입문 친화적입니다.

## (6) Community (커뮤니티)

- Kaggle Forum에서 전 세계 데이터 과학자들과 의견을 교환할 수 있습니다.
- 논문, 코드, 접근법, 팁 등을 공유합니다.
- 상위권 유저(Kaggle Grandmaster)들이 직접 올리는 고급 노트북을 학습 자료로 활용할 수 있습니다.

## (7) Kaggle 등급 체계

> 활동 실적에 따라 **랭크(Rank)** 가 부여됩니다.

| `단계` | `필요 조건` |
| :---: | :---: |
| Novice | 시작 단계 |
| Contributor | 활동 (코드/댓글/데이터셋 공유) 시작 |
| Expert | 일정 수준 이상의 활동 |
| Master | 대회 입상 또는 커뮤니티 기여 |
| Grandmaster | 최상위 전문가 (대회 다수 입상자) |

## (8) Kaggle + Google Colab / GitHub 연동

- Kaggle Notebooks은 GitHub 저장소와 쉽게 연동됩니다.
- Google Colab처럼 클라우드 기반이지만, **Kaggle은 대회 제출 환경과 바로 연결**된다는 점이 강점입니다.

## (9) 사용 시작 방법

1. [https://www.kaggle.com](https://www.kaggle.com) 접속
2. Google 계정으로 로그인
3. `Competitions`, `Datasets`, `Code`, `Learn` 메뉴 탐색
4. `Titanic - Machine Learning from Disaster` 대회로 시작해보세요 (입문자용 튜토리얼입니다)

## (10) Kaggle을 통해 얻을 수 있는 것

- 실제 데이터로 모델을 설계하고 검증하는 **실전 경험**
- 데이터 전처리, 시각화, 모델링 스킬 향상
- 전 세계 전문가들의 코드 분석을 통한 학습
- 커리어 포트폴리오로 활용 (면접 시 강력한 어필 가능)
- 상금 및 랭킹으로 인한 명성 확보
