## (1) **NLP란 무엇인가**

- `자연어 처리(NLP)`는 인간이 사용하는 언어(한국어, 영어, 일본어 등)를 컴퓨터가 이해·생성·해석할 수 있도록 하는 기술 분야입니다.
- NLP는 `언어학 + 컴퓨터과학 + 인공지능(특히 머신러닝/딥러닝)`이 결합된 영역입니다.

## (2) NLP의 주요 목표

> 1) **언어 이해(NLU, Natural Language Understanding)**

- 문장의 의미 파악
- 의도(Intent) 분석
- 감정 분석
- 개체명(사람, 장소 등) 인식 등

> 2) **언어 생성(NLG, Natural Language Generation)**

- 문장 생성
- 요약
- 번역
- 대화 생성(ChatGPT 같은 시스템)

## (3) 자연어 처리의 주요 구성 요소

> 형태소 분석

```
문장을 가장 작은 의미 단위(형태소)로 분해
예: "나는 학교에 간다"
→ ["나/대명사", "는/조사", "학교/명사", "에/조사", "간다/동사"]
```

- 한국어는 **교착어**라 형태소 분석이 특히 중요함.

> 품사 태깅 (POS Tagging)

```
예시 문장 : 학교에 간다
→ 학교/NNG (일반 명사)
→ 에/JKB (부사격 조사)
→ 간다/VV+EF (동사 + 어말어미)
```

- 각 단어가 문장에서 어떤 역할(명사, 동사, 조사 등)을 
하는지 자동으로 구분하는 기술입니다.

> 파싱(구문 분석)

- 문장의 문법적 구조 분석
- 트리 구조로 문장을 표현

> 의미 분석(Semantic Analysis)

```
문장의 의미를 컴퓨터가 해석
예: “나는 은행에 갔다” → ‘금융기관(Bank)’인지 ‘강둑(Bank)’인지 문맥 분석 필요
```

> 자연어 이해(NLU)

- 의도(Intent) 분류, 개체 인식(NER), 질의 응답(QA) 등

> 자연어 생성(NLG)

- 문장을 생성하거나 요약, 번역하는 기술
- GPT, 번역기 등은 NLG 기반

## (4) NLP에서 사용하는 주요 모델/알고리즘

> 1) 전통적 NLP (딥러닝 이전)

- 규칙 기반 시스템
- HMM (Hidden Markov Model)
- CRF (Conditional Random Field)
- n-gram 언어 모델

```
문제점: 문맥을 길게 이해하지 못하고, 규칙 구축이 어려움
```

> 2) 딥러닝 기반 NLP

- RNN, LSTM, GRU
	- 문장처럼 순차적 데이터 처리에 강함
	- 긴 문맥을 다루는 한계

- CNN 기반 NLP
	- 텍스트 분류에서 사용

> 3) 트랜스포머(Transformer) 기반 NLP — 현대 NLP의 핵심

- 2017년 구글의 논문 *Attention is All You Need*에서 등장
- 대부분의 SOTA AI가 Transformer 기반

> Transformer 핵심 요소

- Attention (특히 Self-Attention)
- 병렬 처리 가능 → 학습 속도 빠름
- 긴 문맥 처리 우수

> Transformer 기반의 대표 모델

- **BERT** (문장 이해 중심)
- **GPT 시리즈** (문장 생성 중심)
- **T5** (텍스트 입력 → 텍스트 출력)
- **RoBERTa, ALBERT, ELECTRA 등**

## (5) NLP의 주요 응용 분야

> 1) 기계 번역

- 구글 번역
- DeepL 번역
- NMT(Neural Machine Translation) 모델 사용

> 2) 대화형 AI / 챗봇

- ChatGPT
- Siri, Bixby, Alexa 대규모 언어모델(LLM) 기반

> 3) 감정 분석

```
“이 제품 너무 별로임” → 부정
“최고예요!” → 긍정
```

> 4) 텍스트 요약

- 뉴스 요약, 연구 논문 요약 등

> 5) 정보 추출

- 개체명 인식(NER)
- 관계 추출
- 키워드 추출

> 6) 검색 엔진

- 구글 검색
- 질문에 대한 적합한 문서를 찾는 Ranking 기술 (BERT 기반 모델 사용)

> 7) 음성 인식(ASR) / 음성 합성(TTS)

- 텍스트 기반 NLP와 연결

## (6) NLP의 어려움

> 1) 언어 간의 구조적 차이

- 한국어, 일본어, 중국어는 형태소 분석이 필요
- 영어는 분석 방식이 다름

> 2) 문맥, 모호성

- "나는 은행에서 쉬고 있었다" → 어떤 '은행'인가?

> 3) 세계지식 필요

```
“사과를 들고 있는데 떨어뜨렸다”
→ 떨궈진 것은 ‘사과’지 ‘손’이 아님
```
- `상식` 필요

> 4) 긴 문서 처리

- 1000단어 이상의 문맥 유지가 어려움
- 최신 모델은 긴 컨텍스트를 해결 중

## (7) 최신 기술 동향 (2024~2025 기준)

> 1) LLM (Large Language Models)

- GPT-4, GPT-5 계열, Llama, Claude 등
-  인간 수준의 언어 능력을 목표로 발전

> 2) 멀티모달 모델

- 텍스트 + 이미지 + 오디오 + 영상까지 처리
- 예: GPT-4o, Gemini, Claude 3.5 Sonnet

> 3) RAG (Retrieval-Augmented Generation)

- 검색 + LLM 결합
- 더 정확하고 신뢰성 높은 답변 제공

> 4) 에이전트 AI

- AI가 직접 도구 사용, 검색, 분석, 자동화 실행
- 예: AutoGPT, ReAct, Toolformer
